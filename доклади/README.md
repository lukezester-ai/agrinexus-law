# Доклади и проектна документация

Това е официалната папка за всички документи, свързани с проекта `agrinexus-mvp`.

Оттук нататък всички нови анализи, release notes, планове, проверки и технически отчети се записват тук.

## Структура

- `release-notes-YYYY-MM-DD.md` - версии и deployment бележки
- `security-YYYY-MM-DD.md` - security одити, рискове, hardening
- `review-YYYY-MM-DD.md` - code review доклади
- `plan-YYYY-MM-DD.md` - implementation планове
- `incident-YYYY-MM-DD.md` - проблеми и postmortem
- `changelog-short.md` - кратък публичен changelog (по избор)

## Правила за именуване

- Формат на дата: `YYYY-MM-DD`
- Имена: малки букви, тирета, без интервали
- Един документ = една тема

## Политика за обновяване

- При всяка съществена промяна се добавя документ в тази папка.
- При deploy се добавя/обновява release notes файл.
- Документи извън тази папка се избягват, освен ако са част от кодовата документация на framework/library.
